# Nova Construction - Professional Website

A professional, mobile-responsive website for Nova Construction, providing residential remodels, renovations, handyman services, and fix-ups in Southern California (LA & San Diego).

## Pages

- **Home** (`index.html`) - Welcome page with service overview and call-to-action
- **Services** (`services.html`) - Detailed information about all services offered
- **About Us** (`about.html`) - Team information (Obed & Baha) and company story
- **Contact** (`contact.html`) - Contact form and business contact information

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Professional branding and clean UI
- ✅ Contact form integration with Formspree
- ✅ SEO-optimized (robots.txt, sitemap.xml, meta tags)
- ✅ Fast loading with static HTML/CSS/JS

## Deployment

This site is automatically deployed to GitHub Pages. Push changes to the `main` branch and the site updates automatically.

### Contact Form

The contact form is powered by [Formspree](https://formspree.io). Form submissions are sent directly to the configured Formspree endpoint.

To update the form endpoint:
1. Go to Formspree.io and create a new form
2. Update the form `action` attribute in `contact.html` with your Formspree endpoint
